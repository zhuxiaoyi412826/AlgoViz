# 排序算法可视化项目规范

## 1. 项目概述
- **项目名称**: SortViz - 在线排序算法可视化学习平台
- **核心功能**: 通过动态柱状图动画实时展示十大排序算法的执行过程，支持播放控制、速度调节、数据配置
- **目标用户**: 计算机科学学生、算法学习者、面试备考者

## 2. 技术栈
- **前端**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **无框架依赖**: 纯原生实现，确保轻量快速
- **图标**: Lucide Icons (SVG)
- **字体**: Inter, system-ui

## 6. 代码对照区 - 算法伪代码

### 冒泡排序
```
for i = 0 to n-1:
    for j = 0 to n-i-2:
        if arr[j] > arr[j+1]:
            swap(arr[j], arr[j+1])
```

### 快速排序
```
function quicksort(arr, low, high):
    if low < high:
        pivot = partition(arr, low, high)
        quicksort(arr, low, pivot-1)
        quicksort(arr, pivot+1, high)

function partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j = low to high-1:
        if arr[j] <= pivot:
            i++
            swap(arr[i], arr[j])
    swap(arr[i+1], arr[high])
    return i+1
```

## 7. 动画实现规范

### 7.1 CSS动画
- 使用 CSS transitions 实现平滑过渡
- 柱子位置变换使用 transform: translateX()
- 高度变化使用 transition: height
- 颜色变化使用 transition: background-color

### 7.2 动画帧率
- 目标帧率: 60fps
- 使用 requestAnimationFrame 优化渲染

### 7.3 动画队列
- 记录每一步操作 (比较、交换、移动)
- 支持暂停、继续、单步执行
